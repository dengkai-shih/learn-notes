#### Error Invalid Download Directory
#### Using the Terminal (Recommended)
You need to connect the Snap's ```removable-media``` interface to allow it access to external drives Using get-iplayer snap version.... Open your terminal and run:
```sh
# jdownloader2
sudo snap connect jdownloader2:removable-media
```
```sh
# qbittorrent
sudo snap connect qbittorrent:removable-media
```
